	_Add Dynamic Attributes:

	
		_General:

			1_se abrirá el panel "Add Attribute particleShape1":

				Add Attribute particleShape1/New:
				_________________________________

				en este panel podemos crear atributos personalizados con diferente tipos

				tipos de datos, visibles, ocultos, animables y no animables, así, como 

				con rangos de acción definidos por nosotros


			2_también podemos seleccionar entre atributos avanzados preestablecidos de las partículas

				Add Attribute particleShape1/Particle:
				______________________________________

				_interminados


			3_también podemos seleccionar entre atributos avanzados preestablecidos de controles

				Add Attribute particleShape1/Control:
				______________________________________

				_interminados





		_Opacity:


			1_Add Per Object Attribute:

				agregar atributo de opacidad a todo el sistema de partículas


			2_Add Per Particle Attribute:

				agregar atributo de opacidad por partícula de forma individual


			_Add Attribute: 

				agregar el tipo de atributo seleccionado


			_Cancel: 

				cancelar la operación


		1_Así, al agregar "Add Per Object Attribute:", se agregarán a las opciones de

		"particle1/particlShape1/Render Attributes", las opciones de:


			_Opacity


		y a través de éstos podremos varias la opacidad de las partículas para su renderizado



		2_más si agregamos un atributo "Add per Particle Attribute", se agregará el atributo

		"Opacity PP", ó (Opacidad RGB, por partícula), en la sección "particle1/particleShape1

		/per Prticle (Array) Attributes/Opacity PP", así al hacer "mouse right click", sobre 

		estas opciones podremos crear expresiones como una "ramp", de valores de alpha, si

		elegimos crear un ramp, se agregará el tag "ramp1", a las tags, de "particle1",

		así en la tab, "ramp", podremos editar las opciones de alpha matte en degradado de ramp






		_Color:

			1_Add Per Object Attribute:

				agregar atributos de color a todo el sistema de partículas


			2_Add Per Particle Attribute:

				agregar atributos por partículas de forma individual


			3_Shader: 

				agrega atributos al sistema de partículas con shader/cloud


			_Add Attribute: 

				agregar el tipo de atributo seleccionado


			_Cancel: 

				cancelar la operación


		1_Así, al agregar "Add Per Object Attribute:", se agregarán a las opciones de

		"particle1/particlShape1/Render Attributes", las opciones de:


			_Color Red 

			_Color Green 

			_Color Blue

		y a través de éstos podremos varias el color de las partículas para su renderizado



		2_más si agregamos un atributo "Add per Particle Attribute", se agregará el atributo

		"RGB PP", ó (color RGB, por partícula), en la sección "particle1/particleShape1

		/per Prticle (Array) Attributes/RGB PP", así al hacer "mouse right click", sobre 

		estas opciones podremos crear expresiones como una "ramp", de valores de color, si

		elegimos crear un ramp, se agregará el tag "ramp1", a las tags, de "particle1",
