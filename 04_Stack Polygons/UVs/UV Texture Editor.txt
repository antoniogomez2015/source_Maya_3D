UV Texture Editor:
__________________

Tutorial: https://www.youtube.com/watch?v=dk4eqg1iMLc&index=23&list=PLA9A5E4BDF49EB336


1_seleccionamos edges y vamos ha: Polygons/Sew UV Edges, esta opción permite unir ó fusionar edges para conformar inslands


2_seleccionamos edges y vamos ha: Polygons/Move and Sew UV Edges, esta opción permite mover las inslans 

y unir los edges que las relacionan


3_seleccionamos edges y vamos ha: polygons/split UVs, esta opción permite separar en dos ó más edges para crear islands


4_para organizar las inlands de forma más ediciente en el espacio de trabajo, seleccionamos las inlands deseadas y luego vamos ha: polygons/Layout


5_para desentramar ó suavisar la forma de los uvs: seleccionamos los uvs, deseados y vamos ha: Polygons/Ufold


6_para crear más de un set, para Uvs: Polygons/Create Empty UVs Set, así, podremos copiar y pegar inslands, uvs de un set a otro


	6.1_para copiar y pegar UVs, entre diferentes UVs sets, seleccionamos los Uvs, deseados en el pane UV Texture Editor

	luego vamos ha: Polygons/Copy UVs to UV Set/Set desedo...., así, la selección se pegará en el uv set seleccionado

	6.2_luego seleccionamos esos mismos uvs y le asignamos un nuevo material, luego aplicamos al paso siguiente "Paso 7"


7_para asigner diferentes set, de uvs, a un material: hacemos mouse right click/UV Set/UV Linking, así, se abrirá

el panel "Relationshio Editor", donde podremos seleccionar el UV Set deseado, de esta manera podemos asignar diferentes materiales a una mesh,

y a su vez asignarle diferentes UV Set, por ej: un UV Set al rostro y otro al body...
